About the Project

This project aims to leverage cutting-edge technology in satellite and aerial imagery to enhance emergency management and disaster response efforts. By integrating machine learning models, we can detect and analyze disaster impacts in real-time, providing crucial information to responders and decision-makers.

Project Phases

Phase 1: Disaster Classification

In this phase, we classify the type of disaster using satellite and aerial imagery. The model can identify various types of disasters such as fires, floods, and collapsed buildings.

Phase 2: Fire Damage Estimation

Once a fire is detected, this phase involves estimating the damage caused by the fire. The model calculates the CO2 emissions, total burned area, and provides a detailed fire damage map.

Phase 3: Flood Damage Estimation

In case of a flood, this phase estimates the affected area and provides detailed maps showing the extent of the flooding and the potential impact on the region.

Key Features

  • Real-time monitoring of disaster-affected areas
  • Advanced image processing using machine learning algorithms
  • Accurate detection and classification of various disaster types
  • Integration with existing emergency management systems
  • Detailed damage estimation maps for fires and floods

Technology Stack
